eScooterRadar.com

Website of Eric M.

This was a fun learning experience described in my University of California application.


To help build my computer skills, I created a project that contains as many hardware and software facets as possible. I started off by focusing on having fun and learning as much as possible, but ended up with something that might be useful. The final result is an electric scooter containing a mobile computer with GPS and “radar” and environmental sensors plus software required to make it all work together and interface with the outside world through the web and iPhone. Each component was individually wired and tested.
